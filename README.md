# Student-Database-Management-System
Student Database Management System (SDMS) is a terminal-based Python application that performs CRUD operations on student records. It allows users to add, view, search, update, and delete entries using a menu-driven interface. Data is stored in memory during execution and resets when the program closes, focusing on core logic and data handling.

Features:

1. Add student records

2. View all students

3. Search by ID

4. Update student information

5. Delete records

6. Input validation and user prompts

This project was built to practice core programming fundamentals:

1. CRUD operations

2. Lists and data structures

3. Functions and program flow

4. User input handling

5. Logical problem solving

The focus is on backend logic rather than GUI or database integration.

How to Run

Clone the repository

Open terminal in project folder

Run the program

python sdms.py
Limitations

No persistent storage (data resets after exit)

No graphical interface
